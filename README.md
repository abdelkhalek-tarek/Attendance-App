# Attendance App (Dockerized)

A simple Attendance Management System built for learning and practicing DevOps concepts using Docker, Docker Compose, and containerized services.

---

## 🚀 Project Overview

This project simulates a basic attendance system where users can be managed and attendance records can be tracked.  
The application is fully containerized to simulate a real-world deployment environment.

---

## 🧱 Tech Stack

- Backend: Python (Flask)
- Database: MySQL
- Containerization: Docker
- Orchestration: Docker Compose
